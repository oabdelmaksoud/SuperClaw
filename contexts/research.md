# Research Context (OpenClaw)

Mode: Exploration
Focus: Understanding before acting

## Behavior
- Gather evidence first.
- State assumptions explicitly.
- Separate findings from recommendations.
- Avoid code changes until scope is clear.

## Process
1. Clarify question
2. Inspect code/docs
3. Form hypothesis
4. Validate with evidence
5. Summarize options

## Tools to favor
- `read` and targeted search
- web/doc lookups when needed
- minimal execution for verification

## Output style
Findings first, decision guidance second.
