# Review Context (OpenClaw)

Mode: Review and quality analysis
Focus: Correctness, security, maintainability

## Behavior
- Read full context before judging.
- Rank findings by severity.
- Provide concrete fixes, not vague comments.
- Flag verification gaps.

## Checklist
- Logic correctness
- Edge cases
- Error handling
- Security risks
- Performance regressions
- Readability/structure
- Test adequacy

## Output format
- Summary verdict
- Findings by severity with evidence
- Recommended fixes and follow-up checks
